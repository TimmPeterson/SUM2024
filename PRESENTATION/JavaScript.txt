Для описания взаимодействия клиента с сайтом требуется быстрый и гибкий язык программирования.

JavaScript идеально подходит на роль такого языка:
  1) Во первых JavaScript - это интерпретируемый и динамически типизированный язык.
  2) Во вторых JavaScript совмещает в себе две современные языковые парадигмы:
       - Объектно ориентированность
       - Функциональность
    Другими словами любая переменная в JavaScript является 
    с одной стороны объектом класса, а с другой стороны функцией.

В JavaScript существует восемь встроенных типов:
  1) String    — строка.    
  2) Number    — число.
  3) BigInt    — число произвольной длины.
  4) Boolean   — логическое значение.
  5) Undefined — тип неинициализированной переменной.
  6) Null
  7) Symbol    — используют для создания уникальных идентификаторов в объектах.
  8) Object    — объект.

Для представления объектов в JavaScript используется специальная нотация JSON(JavaScript Object Notation)
Свойства объекта в JSON, называют словом "property". 

Каждый property может содержать одно из следующих значений:
  1) Строка
  2) Число
  3) Объектно
  4) Массив
  5) Истина
  6) Ложь
  7) Null
